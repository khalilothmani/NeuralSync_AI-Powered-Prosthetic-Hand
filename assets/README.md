## assets/ placeholder

Place your project images and photos in this folder:

- `banner.png`         — Main README banner (header image)
- `circuit_photo.jpg`  — Photo of the assembled circuit
- `hand_open.jpg`      — Photo of the hand in open (default) position
- `hand_power.jpg`     — Photo of the power grab in action
- `hand_pinch.jpg`     — Photo of the pinch grab
- `hand_sensitive.jpg` — Photo of the sensitive grab (e.g. holding an egg)
- `demo_mode1.gif`     — GIF demonstration of Mode 1
- `demo_mode2.gif`     — GIF demonstration of Mode 2 (AI vision)

You can reference images in the README like this:

```markdown
![Hand performing power grab](assets/hand_power.jpg)
```
